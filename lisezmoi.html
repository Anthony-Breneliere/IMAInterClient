Instructions d'installation

Installation de node js

Installation de la derniere version de nodeJS. La dernière version installée de npm est la 3.10.3.

Mise en place du proxy au niveau global:

npm config set proxy http://ep.threatpulse.net:80/

Installation de typings, module permettant la compilation des fichiers typescript .ts en fichier javascript .js.

Copie du fichier .typingsrc à la racine du projet:

proxy=http://ep.threatpulse.net:80
rejectUnauthorized=false

Installation du gestionnaire de package jspm

C'est une gestionnaire de package pour le gestionnaire de module SystemJS (celui installé de base avec le projet angular2-quickstart, alternative au gestionnaire de module WebPack).

Instruction d'installation au niveau global:

npm install jspm -g

Instruction d'installation au niveau projet:

cd my-project
npm install jspm --save-dev
jspm init

Configuration du proxy

Elle se fait pas l'intermédiaire de la viable d'envronnement http_proxy:

set HTTP_PROXY=http://ep.threatpulse.net:80
