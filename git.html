Configuration GIT

Prérequis

Installer GIT en local

Git.exe doit être dans le PATH système.

Vérifier qu'aucun proxy n'est configuré pour GIT:

git config --global -l

La configuration globale ne doit pas contenir http.proxy:

Supprimer la configuration du proxy:

git config --global --unset -l

Supprimer la vairable d'environnement HTTP_PROXY.

Procédure de création d'un projet sous Bonobo

Sur le serveur Bonobo

  • Se connecter sur http://svdepotde01-pic/Bonobo.Git.Server/Repository/Index avec son login/password
  • Cliquer créer un nouveau dépôt.
  • Indiquer le nom du dépôt sans underscore, par exemple IMAInter.
  • Sélectionner les contricuteur et cliquer Créer.
  • Faire un copier coller de l'adresse du dépôt, par exemple http://svdepotde01-pic/Bonobo.Git.Server/IMAInter.git.

En local

Se déplacer en ligne de commande dans le dosser contenant les sources.

Créer un fichier .gitignore dans le dossier du projet. Exemple:

#ignore thumbnails created by windows
Thumbs.db

#Ignore files build by Visual Studio
node_modules/**
app/**/*.js
app/**/*.js.map

Initialiser le déopît GIT local:

git init

Cette commande a créé le dossier .git. Ajouter tous les fichiers au dépôt local :

git add *

Spécifier le dépôt GIT distant créé sur le serveur Bonobo:

git remote add origin http://svdepotde01-pic/Bonobo.Git.Server/IMAInter.git

Pousser les modification sur le serveur distant.

git push --set-upstream origin master

Par la suite

On commit et on pousse de cette manière:

git commit -m "(modifications, tâches)"
git push
